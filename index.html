TsBasics

Environment setup

Install typescript globally

npm install -g typescript

Strong typing

Variable types include 

                            any          const number:any = 5;
                            number       const age:number = 23;
                            string       const fName: string = 'John';
                            string array const friends:string[]=['James', 'Jane', 'Jade'];
                            any array    const descr:any[] = [60, 'brown', 5.5, 'tall']; 
                            boolean      const qualified:boolean = true;

Enumerations

Enums allow us to define a set of named constants 

                                    enum color {Red, Green, Blue};
                                    console.log(color.Red);

Type assertion

It's a way of telling typescript what type of variable it is at compile-time. useful to access intellisense 

                                const message:any = "the sentence tests the length property";
                                let lengthOfMessage = (message as string).length;

Custom types

you can assign variable types to almost anything in typescript. 

                                let drawPoint = (point: {x: number, y:number}) => { point.x + point.y }
                                drawPoint({
                                    x: 1,
                                    y: 2
                                })

Interfaces

It is a collection of abstract methods. 

                                    interface Point{
                                        x: number;
                                        y: number;
                                    }
                                    
                                    let drawPoint = (point:Point) =>{
                                        point.x + point.y;
                                    }

Classes

It is an extensible program-code-template for creating objects, providing initial values for state (member variables) and implementations of behavior (member functions or methods) 

                                    class PointClass{
                                        x: number = 12;
                                        y: number = 2;
                                        draw(){
                                            this.x + this.y;
                                        }
                                    }

Constructors

constructors are special member functions which are created when the object is created or defined and its task is to initialize the object of its class. It is called constructor because it constructs the values of data members of the class. 

                                    class PointClass {
                                        constructor(public x?: number, public y?: number) {}
                                        drawPoint() {
                                            this.x + this.y;
                                        }
                                    }

Access modifiers

Access modifiers (or access specifiers) are keywords in object-oriented languages that set the accessibility of classes, methods, and other members 

                                    class PointClass {
                                        constructor(private x: number, private y: number) {}
                                        drawPoint() {
                                            this.x + this.y;
                                        }
                                    }
